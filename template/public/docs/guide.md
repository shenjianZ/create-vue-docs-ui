# Getting Started

## 开始使用 Vue Docs UI

欢迎使用 Vue Docs UI！这是一个基于 Vue 3 的现代文档网站构建工具。

## 快速概览

Vue Docs UI 让您能够：
- 🚀 **快速上手** - 只需几行代码即可创建美观的文档网站
- 📱 **响应式设计** - 完美适配所有设备
- 🎨 **高度可定制** - 灵活的主题和样式配置
- 📝 **Markdown 支持** - 使用熟悉的 Markdown 语法
